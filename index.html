Библиотека промптов

Промпт для объяснения темы

Универсальный запрос

Дидактика
Ты — учитель физики. Объясни тему '[ТЕМА]' девятиклассникам в формате короткого текста (не более 200 слов). Используй одну бытовую аналогию.

Промпт для генерации задач

Разноуровневые задачи

Дидактика
Ты — методист по физике. Сгенерируй 4 расчётные задачи по теме '[ТЕМА]' для [КЛАСС] класса: 2 базовых, 1 повышенный, 1 олимпиадный.

Промпт для плана урока

Технологическая карта

Проектирование
Ты — учитель физики высшей категории. Разработай технологическую карту урока для [КЛАСС] класса по теме '[ТЕМА]'.

Промпт для «крючка внимания»

Проблемные ситуации

Мотивация
Придумай 3 варианта «крючка внимания» для урока физики в [КЛАСС] классе по теме '[ТЕМА]'.

Промпт для Web-кодинга

Интерактивная модель

Программирование
Действуй как Senior Web-разработчик. Напиши код интерактивной физической симуляции на HTML/CSS/JS. Тема: [ТЕМА].
